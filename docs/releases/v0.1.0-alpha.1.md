# Rustorrent v0.1.0-alpha.1

This is the first public macOS preview build of Rustorrent.

Disclaimer: this is a very early prerelease. Expect rough edges, missing polish, incomplete behavior, and bugs that can interrupt downloads or require a restart. It should not be treated as a production-ready BitTorrent client yet, and it should not be used as the only copy of important data.

Platform scope: this release is for Apple Silicon Macs only (`arm64`). It is not intended for Intel Macs, and no `x86_64` build is included in this release.

Packaging scope: this GitHub release ships only the DMG. The app currently embeds the local UI and launcher, but packaging and distribution are still in an early state. Depending on local macOS security settings, you may need to explicitly allow the app to run.

What is included:

- A single macOS DMG containing `Rustorrent.app`
- Native macOS launcher window
- Embedded local UI flow
- Current downloader, seeding, and library management behavior as implemented on the `master` branch at release time

What is not promised in this release:

- Backward compatibility between future preview builds
- Stable settings, session, or resume behavior across all edge cases
- Intel Mac support
- Final signing / notarization expectations
